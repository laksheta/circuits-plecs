# Calculation
## PI Controller
$$K_p = \frac{\tau \omega_c}{6Dc}, \quad K_i = \frac{K_p}{\tau}$$

 With:

$$G_{DL} = \frac{R_{load}}{2V_0} \cdot \frac{V_i V_s(1-aD)}{a f_{sw} L}$$

$$\tau = \frac{C R_{load}}{a}$$

$$\omega_c = 2\pi f_{bw}$$
