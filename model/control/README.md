## PI Controller
$$K_p = \frac{\tau \omega_c}{G_{dc}}, \quad K_i = \frac{K_p}{\tau}$$

 With:

$$G_{dc} = \frac{R_{load}}{2V_0} \cdot \frac{V_i V'_2(1-2D)}{2 f_{sw} L}$$

$$\tau = \frac{C R_{load}}{2}$$

$$\omega_c = 2\pi f_{bw}$$
